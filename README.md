# VR Teleoperator for Boston Dynamics Spot

Tested on:
- Ubuntu 22.04
- Python 3.10 --> only works with LeRobot Version <= 4.4 (since newer versions use pyhton 3.12 --> not compatible with Spot-Api)


## Dependencies

Teleoperator for LeRobot Boston Dynamics Spot implementation:

https://github.com/J0STR/lerobot_robot_spot

With Oculus Quest 2/3. Use this Godot APP:

https://github.com/J0STR/Godot_VR_App

## Mapping

Right VR Controller:
- Control Forward/Backward Movement
- Control Left/Right Movement

Left VR Controller:
- Press A + Controller Movement --> Arm Movement
- Triggers open and close the Gripper
